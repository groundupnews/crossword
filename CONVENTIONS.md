# Conventions for this project

- Whenever updating css or javascript files, consider adding a query string ?v=n+1 in the HTML file where the static resource is used. This makes testing much easier for me because my browser will definitely reload the static resource.

# Principles

- This is not vibe-coded. The design will be fully specified by me. I will review all code generated by AI.
- We are not in a rush. One piece of code to be generated at a time and then reviewed and refined by me.
- Less code is better.
- Tests should be thought through and assist, not drive development.
- Value simple solutions over complex ones.
